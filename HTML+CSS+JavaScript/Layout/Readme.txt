常用的网页布局结构
{
    单行单列布局
    {
        设定宽度
        {
            使用带单位的数字值;
            使用父元素(浏览器窗口)宽度的百分比;
        }
        一般在浏览器中居中显示
        {
            code(CSS)
            {
                margin: 0 auto;
            }
        }
        div宽度
        {
            同上;
        }
    }
    多列式布局
    {
        利用position定位属性实现多列式固定布局
        {
            一般让子元素绝对定位(不占空间), 父元素相对定位(不失去空间)但不设偏移量, 以保证子元素依据父元素定位;
        }
        利用float浮动属性实现多列式自适应布局
        {
            多列子元素应同时浮动, 父元素(空元素)可以同时浮动自适应子元素高度, 但是必须考虑后续元素, 也可以不浮动, 但是必须设置高度防止后续元素上移;
        }
    }
}
